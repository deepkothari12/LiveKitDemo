LIVEKIT MEETING TRANSCRIPTION - QUICK START
===========================================

1. INSTALL DEPENDENCIES
   pip install -r requirements.txt

2. CONFIGURE .ENV FILE
   - Add LiveKit credentials
   - Add AWS S3 credentials
   - Add Colab API URL

3. START COLAB SERVER
   - Open Google Colab
   - Copy code from FIXED_COLAB_CODE.py
   - Run notebook
   - Copy ngrok URL to .env (COLAB_API_URL)

4. START SERVER
   python server.py

5. OPEN BROWSER
   http://localhost:3000

6. TEST
   - Start meeting
   - Start recording
   - Talk for 30-60 seconds
   - Stop recording
   - Check cleaned-audio/ folder
   - Wait for transcript in S3

FOLDER STRUCTURE
================

Essential Files:
  server.py                      - Main server
  speaker_diarization_handler.py - Speaker separation
  audio_cleaner.py               - Audio preprocessing
  index.html                     - Frontend UI
  requirements.txt               - Dependencies
  .env                           - Configuration
  FIXED_COLAB_CODE.py           - Colab server code

Output Folders:
  cleaned-audio/                - Cleaned audio files (local)

Logs:
  livekit_server.log            - Server events
  webhook_events.log            - Webhook events

TROUBLESHOOTING
===============

No transcription?
  - Check Colab is running
  - Verify COLAB_API_URL in .env
  - Check livekit_server.log

Audio cleaning fails?
  - Install: pip install librosa soundfile noisereduce
  - Check logs for errors
  - Falls back to original audio

S3 issues?
  - Verify AWS credentials
  - Check bucket permissions
  - Ensure correct region

PERFORMANCE
===========

Audio Cleaning:
  1-min recording: ~5 seconds
  5-min recording: ~15 seconds

Transcription:
  10-20 seconds per participant

Total Pipeline:
  ~60-90 seconds for 1-minute recording
